interface struct 能否相互嵌套

  1. struct struct //继承(不能多态), 如果内部struct实现了接口, 它也相当于实现了接口
  2. struct interface //可以多态
  3. interface interface //单纯的导入
  4. interface struct //不允许

实际/动态类型和静态类型

  对于接口类型的变量f来说，Formatter是它的静态类型，并且这是永远不会变的。此时它是没有动态类型。他的动态类型会随着赋给它的动态值而变化，当将一个*Person类型的变量赋值给f时,f的动态类型（也叫实际类型）就是*Persion。

type Formatter interface {  
    Format() string  
}  
type Person struct {  
    name string  
}  
  
func (p *Person) Format() string {  
    return fmt.Sprintf("%s is person", p.name)  
}  
  
type Animals struct {  
    name string  
}  
  
func (a *Animals) Format() string {  
    return fmt.Sprintf("%s is animals", a.name)  
}  
  
func main() {  
    var f Formatter  
    f = &Person{name: "axlrose"}  
    f = &Animals{name: "axlrose"}  
    f.Format()
}
